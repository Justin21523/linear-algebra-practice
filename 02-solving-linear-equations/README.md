# 第二章：解線性方程組 (Solving Linear Equations)

> Ax = b — 線性代數最核心的問題

## 本章概述

「給定矩陣 A 和向量 b，求解向量 x 使得 Ax = b」是線性代數最基本也最重要的問題。本章介紹系統化的解法。

## 為什麼重要？

線性方程組無所不在：

| 應用領域 | 範例 |
|----------|------|
| **工程模擬** | 結構分析、流體力學（有限元素法）|
| **電路分析** | 求解節點電壓（克希荷夫定律）|
| **機器學習** | 線性迴歸、最小平方法 |
| **電腦圖學** | 座標變換、光線追蹤 |
| **經濟模型** | 投入產出分析、均衡價格 |

## 本章單元

```
01-gaussian-elimination/       高斯消去法（核心演算法）
02-lu-decomposition/           LU 分解（高效率的系統化方法）
03-inverse-matrix/             反矩陣（理論重要，實務少用）
04-computational-complexity/   計算複雜度（效能分析）
```

## 學習目標

完成本章後，你應該能夠：

1. **高斯消去法**
   - 執行消去步驟將矩陣化為上三角形式
   - 進行回代 (back substitution) 求解
   - 處理需要換列的情況（pivoting）
   - 理解消去矩陣 E 的作用

2. **LU 分解**
   - 理解 A = LU 的意義
   - 手動計算 L 和 U
   - 利用 LU 分解高效求解多個右手邊 b

3. **反矩陣**
   - 判斷矩陣是否可逆
   - 用高斯-乔丹消去法求反矩陣
   - 理解 A⁻¹ 存在的條件

4. **計算複雜度**
   - 分析高斯消去法的 O(n³) 複雜度
   - 理解為什麼「求反矩陣再乘」不是好方法

## 關鍵概念

### 矩陣方程 Ax = b

```
[a₁₁  a₁₂  a₁₃] [x₁]   [b₁]
[a₂₁  a₂₂  a₂₃] [x₂] = [b₂]
[a₃₁  a₃₂  a₃₃] [x₃]   [b₃]
```

### 三種可能的結果

| 情況 | 條件 | 解的個數 |
|------|------|----------|
| **唯一解** | A 可逆（滿秩）| 1 |
| **無解** | b 不在 A 的行空間 | 0 |
| **無限多解** | A 不滿秩，b 在行空間 | ∞ |

### 增廣矩陣 (Augmented Matrix)

```
[A | b] = [a₁₁  a₁₂  a₁₃ | b₁]
          [a₂₁  a₂₂  a₂₃ | b₂]
          [a₃₁  a₃₂  a₃₃ | b₃]
```

## 關鍵公式速查

```
高斯消去：    [A|b] → [U|c]    （上三角形式）
LU 分解：     A = LU           （下三角 × 上三角）
反矩陣：      AA⁻¹ = A⁻¹A = I
求解步驟：    Ax = b → Ly = b → Ux = y

回代公式：    xₙ = cₙ/uₙₙ
             xᵢ = (cᵢ - Σⱼ₌ᵢ₊₁ⁿ uᵢⱼxⱼ) / uᵢᵢ
```

## 考試重點

### 常見題型

1. **消去法計算**：給定矩陣，執行高斯消去
2. **LU 分解**：求 L 和 U
3. **反矩陣計算**：用高斯-乔丹法求 A⁻¹
4. **解的判斷**：判斷有幾個解

### 關鍵觀念

- **pivoting（選主元）** 的必要性
- 為什麼 LU 分解比每次重新消去更有效
- 反矩陣何時存在（det(A) ≠ 0）
