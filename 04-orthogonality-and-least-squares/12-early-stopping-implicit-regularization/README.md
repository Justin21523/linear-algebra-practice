# Early Stopping（隱式正則化）：用梯度下降的「迭代次數」控制泛化

> 你不一定要加 λ 才叫正則化：對病態最小平方，提早停止迭代也能抑制噪聲放大

## 學習目標

完成本單元後，你應該能：

1. 用 GD 解不正則化最小平方：`min_x 1/2‖Ax-b‖²`
2. 把「迭代次數 T」當作超參數，像選 `λ/k` 一樣用驗證集選 `T`
3. 觀察典型現象（尤其在病態/高維特徵）：
   - train RMSE 隨迭代下降
   - val RMSE 常呈現 U 形（太少欠擬合、太多過擬合/噪聲放大）
   - `‖x‖` 隨迭代變大（係數逐步長大）
