# 基底與維度 (Basis and Dimension)

> 向量空間的「座標系統」與「大小」

## 1. 基底 (Basis)

### 定義

向量空間 V 的**基底** (basis) 是一組向量 {v₁, v₂, ..., vₙ}，滿足：

1. **線性獨立**：向量組互不冗餘
2. **生成 V**：span{v₁, ..., vₙ} = V

### 等價說法

```
B 是 V 的基底 ⟺ V 中每個向量都能被 B 唯一地線性組合表示
```

### 範例

```
ℝ² 的標準基底：{[1,0], [0,1]}
ℝ³ 的標準基底：{[1,0,0], [0,1,0], [0,0,1]}
Pₙ 的標準基底：{1, x, x², ..., xⁿ}
```

---

## 2. 維度 (Dimension)

### 定義

向量空間 V 的**維度** dim(V) 是 V 的任一基底所含向量的個數。

### 重要定理

```
同一向量空間的所有基底都有相同的元素個數
```

這就是為什麼「維度」是良好定義的。

### 常見維度

| 空間 | 維度 |
|------|------|
| ℝⁿ | n |
| Pₙ（≤n 次多項式）| n+1 |
| M_{m×n}（m×n 矩陣）| m×n |
| 零空間 | 0 |

---

## 3. 標準基底 vs 非標準基底

### 標準基底

```
ℝⁿ 的標準基底：e₁, e₂, ..., eₙ
其中 eᵢ = [0, ..., 0, 1, 0, ..., 0]（第 i 位是 1）
```

### 非標準基底

```
ℝ² 的另一組基底：{[1,1], [1,-1]}

驗證：
1. 獨立：det [1  1] = -2 ≠ 0 ✓
          [1 -1]

2. 生成 ℝ²：任意 [a,b] = c₁[1,1] + c₂[1,-1]
   解得 c₁ = (a+b)/2, c₂ = (a-b)/2 ✓
```

### 選擇基底的考量

- **標準基底**：計算簡單，直觀
- **正交基底**：投影、分解方便
- **特徵向量基底**：矩陣變成對角形式

## 4. 座標 (Coordinates)

### 定義

給定基底 B = {v₁, ..., vₙ}，向量 x 的 **B-座標** 是：

```
[x]_B = [c₁, c₂, ..., cₙ]ᵀ

其中 x = c₁v₁ + c₂v₂ + ... + cₙvₙ
```

### 範例

```
B = {[1,1], [1,-1]}（ℝ² 的非標準基底）
x = [3, 1]

求 [x]_B：
[3, 1] = c₁[1,1] + c₂[1,-1]
c₁ + c₂ = 3
c₁ - c₂ = 1
⟹ c₁ = 2, c₂ = 1

[x]_B = [2, 1]ᵀ
```

### 座標變換

若 P = [v₁ | v₂ | ... | vₙ]（基底向量排成矩陣），則：

```
x = P · [x]_B
[x]_B = P⁻¹ · x
```

## 5. 找基底的方法

### 給定生成集，找基底

1. 將向量排成矩陣的行
2. 做 RREF
3. 主元行對應的原向量是基底

### 給定子空間定義，找基底

**行空間 C(A)**：RREF 的主元行對應的原矩陣的行

**零空間 N(A)**：對每個自由變數設 1，其他自由變數設 0，回代

---

## 6. 秩-零度定理（再強調）

### 定理

對於 m×n 矩陣 A：

```
dim C(A) + dim N(A) = n

rank(A) + nullity(A) = n
```

### 直覺

```
n 維輸入空間 = 「有效維度」+ 「被壓縮到零的維度」
            = rank        + nullity
```

---

## 7. 基底的性質

### 向量數等於維度

```
B 是 n 維空間的基底 ⟺ |B| = n 且（獨立 或 生成）
```

### 子空間的維度

```
若 W 是 V 的子空間，則 dim(W) ≤ dim(V)
若 dim(W) = dim(V)，則 W = V
```

### 維度的加法公式

```
dim(U + W) = dim(U) + dim(W) - dim(U ∩ W)
```

## 9. 考試重點

### 常見題型

1. **找基底**：給定向量組，找出一組基底
2. **計算維度**：給定子空間，求維度
3. **座標變換**：在不同基底下求座標
4. **秩-零度定理應用**：已知部分資訊，求其他

### 關鍵公式

```
dim ℝⁿ = n
dim Pₙ = n + 1
dim M_{m×n} = m × n
dim C(A) = rank(A) = 主元數
dim N(A) = n - rank(A) = 自由變數數
```

### 常見錯誤

- 混淆基底的「元素個數」和「向量維度」
- 誤以為任意 n 個向量都是 ℝⁿ 的基底
- 座標變換矩陣的方向搞錯
