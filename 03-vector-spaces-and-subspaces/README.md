# 第三章：向量空間與子空間 (Vector Spaces and Subspaces)

> 從具體計算到抽象結構——理解線性代數的核心概念

## 本章概述

向量空間是線性代數的核心抽象。理解向量空間讓我們能夠：
- 用統一的框架處理各種「線性」問題
- 理解線性方程組解的結構
- 為後續的特徵值、SVD 等主題打下基礎

## 為什麼重要？

| 概念 | 實際意義 |
|------|----------|
| **行空間 (Column space)** | 線性方程組 Ax = b 有解的條件 |
| **零空間 (Null space)** | 齊次方程組 Ax = 0 的解集合 |
| **維度 (Dimension)** | 描述資料的「自由度」|
| **基底 (Basis)** | 最精簡的表示方式 |
| **秩 (Rank)** | 矩陣的「有效資訊量」|

## 學習目標

完成本章後，你應該能夠：

1. **向量空間**
   - 驗證一個集合是否為向量空間
   - 理解子空間的概念
   - 識別常見的向量空間（ℝⁿ、多項式、矩陣空間）

2. **行空間與零空間**
   - 找出矩陣的行空間 C(A)
   - 計算零空間 N(A)
   - 理解 Ax = b 有解 ⟺ b ∈ C(A)

3. **線性獨立**
   - 判斷向量組是否線性獨立
   - 理解獨立與相依的幾何意義
   - 找出最大線性獨立子集

4. **基底與維度**
   - 找出向量空間的基底
   - 計算維度
   - 理解秩-零度定理 (Rank-Nullity Theorem)

5. **四大基本子空間**
   - 識別 C(A)、N(A)、C(Aᵀ)、N(Aᵀ)
   - 理解它們之間的關係
   - 理解維度公式：dim C(A) + dim N(A) = n

## 關鍵概念圖

```
                    ℝⁿ
                   ╱   ╲
                  ╱     ╲
            行空間      零空間
            C(A)        N(A)
              │           │
              └─────┬─────┘
                    │
              dim + dim = n
              (秩-零度定理)
```

## 關鍵公式速查

```
行空間：        C(A) = {Ax : x ∈ ℝⁿ}
零空間：        N(A) = {x : Ax = 0}
秩：            rank(A) = dim C(A) = dim C(Aᵀ)
零度：          nullity(A) = dim N(A)
秩-零度定理：   rank(A) + nullity(A) = n（行數）
```

## 考試重點

### 常見題型

1. **判斷子空間**：給定集合，判斷是否為子空間
2. **求零空間**：給定矩陣，求 N(A) 的基底
3. **求行空間**：給定矩陣，求 C(A) 的基底
4. **線性獨立**：判斷向量組是否獨立
5. **維度計算**：求秩、零度
6. **四大子空間**：分析矩陣的四個基本子空間

### 關鍵觀念

- 子空間必須包含零向量
- 零空間的基底就是 Ax = 0 的「特解」
- rank(A) = pivot 的數量
