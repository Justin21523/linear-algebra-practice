# 行列式的性質 (Properties of Determinants)

> 行列式——矩陣最神秘的單一數值

## 1. 什麼是行列式？

### 定義

行列式是將方陣映射到純量的函數：

```
det: n×n 矩陣 → 實數

det(A) 或 |A|
```

### 2×2 行列式

```
    [a  b]
A = [c  d]

det(A) = ad - bc
```

### 3×3 行列式

```
    [a₁  b₁  c₁]
A = [a₂  b₂  c₂]
    [a₃  b₃  c₃]

det(A) = a₁(b₂c₃ - b₃c₂) - b₁(a₂c₃ - a₃c₂) + c₁(a₂b₃ - a₃b₂)
```

---

## 2. 行列式的三大性質

### 性質 1：恆等矩陣的行列式

```
det(I) = 1
```

### 性質 2：列交換

```
交換兩列 → 行列式變號

det(列交換後的 A) = -det(A)
```

### 性質 3：線性性（對單列）

```
對第一列線性：

det([a + a', b, c, ...])   = det([a, b, c, ...]) + det([a', b, c, ...])
det([ka, b, c, ...])       = k · det([a, b, c, ...])
```

---

## 3. 由三大性質推導的結果

### 推論 1：重複列

```
若兩列相同 → det(A) = 0

證明：交換相同列，矩陣不變但行列式變號
det(A) = -det(A) → det(A) = 0
```

### 推論 2：零列

```
若有一列全為 0 → det(A) = 0

證明：0·det(A) = det(0·第i列) = det(A的一列為0)
```

### 推論 3：列運算

```
rᵢ ← rᵢ + c·rⱼ 不改變行列式

證明：利用線性性和重複列性質
```

### 推論 4：上三角矩陣

```
det(上三角矩陣) = 對角線乘積

    [a  *  *]
det [0  b  *] = abc
    [0  0  c]
```

---

## 4. 重要公式

### 乘積公式

```
det(AB) = det(A) · det(B)

重要！矩陣乘法的行列式 = 行列式的乘積
```

### 轉置公式

```
det(Aᵀ) = det(A)

行列式對列和行的性質相同
```

### 逆矩陣公式

```
det(A⁻¹) = 1/det(A)

因為 det(A·A⁻¹) = det(I) = 1
```

### 冪次公式

```
det(Aⁿ) = (det(A))ⁿ
```

### 純量乘法

```
det(cA) = cⁿ det(A)   （n×n 矩陣）

注意：c 乘到每一列，共 n 列
```

---

## 5. 可逆性判定

### 核心定理

```
A 可逆 ⟺ det(A) ≠ 0
```

### 證明思路

```
A 可逆
⟺ A 可列運算化為 I
⟺ det(A) ≠ 0（列運算只改變正負或乘常數）
```

### 奇異矩陣

```
det(A) = 0 稱 A 為奇異矩陣（singular）

- 不可逆
- 行/列線性相關
- rank(A) < n
```

## 6. 計算技巧

### 方法 1：列運算化為上三角

```
1. 用列運算將 A 化為上三角 U
2. det(A) = ±(對角線乘積)
3. 每次列交換變號
```

### 方法 2：展開公式

```
沿第 i 列展開：

det(A) = Σⱼ aᵢⱼ Cᵢⱼ

Cᵢⱼ = (-1)^(i+j) Mᵢⱼ（餘因子）
Mᵢⱼ = 去掉第 i 列第 j 行的子行列式
```

### 方法 3：LU 分解

```
A = LU

det(A) = det(L) · det(U)
       = 1 · (U 的對角線乘積)
```

## 7. 特殊矩陣的行列式

| 矩陣類型 | 行列式 |
|---------|--------|
| 對角矩陣 | 對角線乘積 |
| 三角矩陣 | 對角線乘積 |
| 正交矩陣 | ±1 |
| 旋轉矩陣 | 1 |
| 反射矩陣 | -1 |
| 置換矩陣 | ±1（依奇偶性）|

## 8. 完整範例

### 計算 3×3 行列式

```
    [2  1  3]
A = [0  4  5]
    [0  0  6]

因為是上三角矩陣：
det(A) = 2 × 4 × 6 = 48
```

### 列運算計算

```
    [1  2  3]
A = [4  5  6]
    [7  8  10]

r₂ ← r₂ - 4r₁
r₃ ← r₃ - 7r₁

    [1   2   3 ]
→   [0  -3  -6 ]
    [0  -6  -11]

r₃ ← r₃ - 2r₂

    [1   2   3]
→   [0  -3  -6]
    [0   0   1]

det(A) = 1 × (-3) × 1 = -3
```

## 10. 考試重點

### 常見題型

1. **計算行列式**（2×2, 3×3）
2. **驗證行列式性質**
3. **判斷可逆性**
4. **推導 det(AB) = det(A)det(B)**

### 必背公式

```
2×2：ad - bc
det(AB) = det(A)det(B)
det(Aᵀ) = det(A)
det(A⁻¹) = 1/det(A)
det(cA) = cⁿdet(A)
```

### 常見錯誤

- det(A+B) ≠ det(A) + det(B)
- det(cA) ≠ c·det(A)
- 忘記列交換變號
