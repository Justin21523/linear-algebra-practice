# 第一章：向量與矩陣 (Vectors and Matrices)

> 線性代數的基礎建構模塊

## 本章概述

向量與矩陣是線性代數的核心語言。在進入更抽象的向量空間概念之前，我們需要先熟悉這些基本物件的運算規則與幾何意義。

## 為什麼重要？

| 應用領域 | 向量/矩陣的角色 |
|----------|-----------------|
| **機器學習** | 特徵向量、權重矩陣、資料集表示 |
| **電腦圖學** | 頂點座標、變換矩陣（旋轉、縮放、平移）|
| **物理模擬** | 力、速度、位置向量 |
| **資料科學** | 資料點、相關係數矩陣 |
| **神經網路** | 輸入層、隱藏層的矩陣運算 |

## 本章單元

## 關鍵公式速查

### 向量運算

```
向量加法：    u + v = [u₁+v₁, u₂+v₂, ..., uₙ+vₙ]
純量乘法：    cu = [cu₁, cu₂, ..., cuₙ]
向量長度：    ‖v‖ = √(v₁² + v₂² + ... + vₙ²)
單位向量：    û = v / ‖v‖
內積：        u·v = u₁v₁ + u₂v₂ + ... + uₙvₙ = ‖u‖‖v‖cos(θ)
```

### 矩陣運算

```
轉置：        (Aᵀ)ᵢⱼ = Aⱼᵢ
矩陣乘法：    (AB)ᵢⱼ = Σₖ Aᵢₖ Bₖⱼ
結合律：      (AB)C = A(BC)  ✓
交換律：      AB ≠ BA        ✗ (一般情況)
```

## 考試重點

- **向量內積**：計算、正交判斷（u·v = 0）
- **矩陣乘法**：維度相容性、結果矩陣大小
- **轉置性質**：(AB)ᵀ = BᵀAᵀ
- **特殊矩陣性質**：對稱矩陣 Aᵀ = A
