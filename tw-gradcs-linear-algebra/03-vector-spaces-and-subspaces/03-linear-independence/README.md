# 線性獨立 (Linear Independence)

> 判斷向量組是否「真正」貢獻新的方向

## 1. 定義

### 線性獨立 (Linearly Independent)

向量組 {v₁, v₂, ..., vₖ} 是**線性獨立**的，若且唯若：

```
c₁v₁ + c₂v₂ + ... + cₖvₖ = 0  ⟹  c₁ = c₂ = ... = cₖ = 0
```

**意思**：唯一能讓線性組合等於零的方式是所有係數都為零。

### 線性相依 (Linearly Dependent)

向量組是**線性相依**的，若存在不全為零的係數使得：

```
c₁v₁ + c₂v₂ + ... + cₖvₖ = 0，其中某些 cᵢ ≠ 0
```

**意思**：某個向量可以被其他向量的線性組合表示。

---

## 2. 幾何意義

### 2D 空間

```
線性獨立：兩向量不平行
線性相依：兩向量平行（同方向或反方向）
```

### 3D 空間

```
兩個向量：
  獨立：不平行
  相依：平行

三個向量：
  獨立：不共面
  相依：共面（可能有兩個平行，或三個在同一平面）
```

### 一般情況

```
k 個向量線性獨立 ⟺ 這些向量張成 k 維子空間
k 個向量線性相依 ⟺ 這些向量張成的維度 < k
```

---

## 3. 判斷方法

### 方法一：定義法

解 c₁v₁ + c₂v₂ + ... + cₖvₖ = 0

若只有零解 ⟹ 獨立
若有非零解 ⟹ 相依

### 方法二：矩陣法

將向量排成矩陣的行：A = [v₁ | v₂ | ... | vₖ]

```
向量組獨立 ⟺ N(A) = {0}
           ⟺ Ax = 0 只有零解
           ⟺ rank(A) = k（行數）
```

### 方法三：行列式法（僅限方陣）

若向量個數 = 向量維度：

```
向量組獨立 ⟺ det(A) ≠ 0
```

---

## 4. 重要性質

### 單一向量

```
{v} 獨立 ⟺ v ≠ 0
{0} 相依（因為 1·0 = 0）
```

### 包含零向量

```
若向量組包含零向量，則一定相依
```

### 子集

```
若向量組相依，則任何包含它的更大向量組也相依
若向量組獨立，則任何子集也獨立
```

### 維度限制

```
在 n 維空間中，最多 n 個向量可以獨立
超過 n 個向量一定相依
```

---

## 5. 範例

### 範例 1：兩個 2D 向量

```
v₁ = [1, 2], v₂ = [3, 4]

檢查：c₁[1, 2] + c₂[3, 4] = [0, 0]
      c₁ + 3c₂ = 0
      2c₁ + 4c₂ = 0

det [1  3] = 4 - 6 = -2 ≠ 0
    [2  4]

獨立！
```

### 範例 2：相依向量

```
v₁ = [1, 2], v₂ = [2, 4]

注意：v₂ = 2·v₁

2·v₁ - 1·v₂ = 0（非零解存在）

相依！
```

### 範例 3：三個 3D 向量

```
v₁ = [1, 0, 0], v₂ = [0, 1, 0], v₃ = [1, 1, 0]

注意：v₃ = v₁ + v₂

v₁ + v₂ - v₃ = 0（非零解存在）

相依！（三個向量都在 xy 平面上）
```

## 6. 與矩陣的關係

### 行獨立

A 的行向量獨立 ⟺ rank(A) = n（行數）⟺ N(A) = {0}

### 列獨立

A 的列向量獨立 ⟺ rank(A) = m（列數）⟺ N(Aᵀ) = {0}

### 滿秩

```
A 行滿秩 ⟺ rank(A) = n ⟺ 行獨立
A 列滿秩 ⟺ rank(A) = m ⟺ 列獨立

若 A 是方陣：
  滿秩 ⟺ 行獨立 ⟺ 列獨立 ⟺ det(A) ≠ 0 ⟺ A 可逆
```

## 7. 最大線性獨立子集

### 定義

向量組的**最大線性獨立子集**是：
- 獨立的子集
- 加入任何其他向量就會相依

### 性質

```
所有最大獨立子集的大小相同 = rank
```

### 演算法

1. 將向量排成矩陣的行
2. 做 RREF
3. 主元行對應的原向量形成最大獨立子集

## 9. 考試重點

### 常見題型

1. **判斷獨立/相依**：給定向量組
2. **找相依關係**：找出使線性組合為零的係數
3. **找最大獨立子集**：從向量組中選出獨立的
4. **維度與獨立性**：n 維空間中 k 個向量的獨立性

### 快速判斷

```
ℝⁿ 中：
- n+1 個向量一定相依
- n 個向量：看 det
- 包含零向量：一定相依
- 兩個向量：看是否平行
```

### 常見錯誤

- 混淆「獨立」和「正交」
- 以為不平行就一定獨立（三個以上向量）
- 忘記考慮向量維度的限制
