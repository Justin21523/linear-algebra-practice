# 向量空間的定義 (Vector Space Definition)

> 將「向量」從箭頭推廣到更抽象的概念

## 1. 為什麼需要抽象定義？

### 向量不只是箭頭

以下這些都可以視為「向量」：

| 物件 | 說明 |
|------|------|
| ℝⁿ 中的向量 | (1, 2, 3) — 熟悉的向量 |
| 多項式 | 1 + 2x + 3x² — 係數可以相加、純量乘 |
| 函數 | sin(x), eˣ — 函數可以相加、純量乘 |
| 矩陣 | 2×2 矩陣 — 矩陣可以相加、純量乘 |

它們都滿足相同的運算規則！

---

## 2. 向量空間的定義 (Definition)

### 正式定義

一個**向量空間** V（over 實數 ℝ）是一個集合，配備兩種運算：
1. **向量加法**：V × V → V
2. **純量乘法**：ℝ × V → V

並滿足以下 8 條公理：

### 加法公理（4 條）

| 公理 | 描述 | 公式 |
|------|------|------|
| A1 | 封閉性 (Closure) | u + v ∈ V |
| A2 | 交換律 (Commutativity) | u + v = v + u |
| A3 | 結合律 (Associativity) | (u + v) + w = u + (v + w) |
| A4 | 零向量存在 (Zero vector) | ∃ 0 使得 v + 0 = v |
| A5 | 反向量存在 (Inverse) | ∀v, ∃(-v) 使得 v + (-v) = 0 |

### 純量乘法公理（3 條）

| 公理 | 描述 | 公式 |
|------|------|------|
| M1 | 封閉性 (Closure) | cv ∈ V |
| M2 | 分配律 (Distributivity) | c(u + v) = cu + cv |
| M3 | 分配律 | (c + d)v = cv + dv |
| M4 | 結合律 (Associativity) | c(dv) = (cd)v |
| M5 | 單位元 (Identity) | 1·v = v |

## 3. 常見的向量空間

### ℝⁿ：n 維實向量空間

最基本的向量空間：

```
ℝ² = {(x, y) : x, y ∈ ℝ}      — 平面
ℝ³ = {(x, y, z) : x, y, z ∈ ℝ} — 三維空間
ℝⁿ = {(x₁, ..., xₙ) : xᵢ ∈ ℝ}  — n 維空間
```

### Pₙ：次數 ≤ n 的多項式

```
Pₙ = {a₀ + a₁x + a₂x² + ... + aₙxⁿ : aᵢ ∈ ℝ}

加法：(1 + 2x) + (3 + x) = 4 + 3x
純量乘法：2(1 + 2x) = 2 + 4x
零向量：0 多項式
```

**維度：n + 1**（係數有 n+1 個）

### M_{m×n}：m×n 實矩陣

```
M_{2×2} = 所有 2×2 實矩陣

加法：逐元素相加
純量乘法：每個元素乘以純量
零向量：零矩陣
```

**維度：m × n**

### C[a,b]：[a,b] 上的連續函數

```
C[0,1] = {f : [0,1] → ℝ, f 連續}

加法：(f + g)(x) = f(x) + g(x)
純量乘法：(cf)(x) = c·f(x)
零向量：f(x) = 0
```

**維度：無限維**

## 4. 子空間 (Subspace)

### 定義

向量空間 V 的**子空間** W 是 V 的一個子集，且 W 本身也是向量空間（使用相同的加法和純量乘法）。

### 判斷子空間的簡化條件

W ⊆ V 是子空間 ⟺ 滿足以下三條：

1. **非空**：0 ∈ W（或 W ≠ ∅）
2. **加法封閉**：u, v ∈ W ⟹ u + v ∈ W
3. **純量乘法封閉**：v ∈ W, c ∈ ℝ ⟹ cv ∈ W

### 等價條件（更簡潔）

W 是子空間 ⟺ W 非空，且 ∀u, v ∈ W, ∀c, d ∈ ℝ：cu + dv ∈ W

（線性組合封閉）

---

## 5. 子空間的例子

### ✅ 是子空間

```
1. 通過原點的直線（ℝ² 的子空間）
   W = {t(1, 2) : t ∈ ℝ}

2. 通過原點的平面（ℝ³ 的子空間）
   W = {(x, y, 0) : x, y ∈ ℝ}

3. 零空間（任何向量空間的子空間）
   W = {0}

4. 整個空間（trivial）
   W = V
```

### ❌ 不是子空間

```
1. 不通過原點的直線
   W = {(x, y) : y = x + 1}
   ✗ 不包含零向量 (0, 0)

2. 第一象限
   W = {(x, y) : x ≥ 0, y ≥ 0}
   ✗ 純量乘法不封閉：(1, 1) ∈ W，但 -1·(1, 1) = (-1, -1) ∉ W

3. 整數向量
   W = {(x, y) : x, y ∈ ℤ}
   ✗ 純量乘法不封閉：(1, 0) ∈ W，但 0.5·(1, 0) = (0.5, 0) ∉ W
```

## 6. 生成與張成 (Span)

### 定義

向量組 {v₁, v₂, ..., vₖ} 的**張成** (span) 是所有線性組合形成的集合：

```
span{v₁, ..., vₖ} = {c₁v₁ + c₂v₂ + ... + cₖvₖ : cᵢ ∈ ℝ}
```

### 重要性質

- span{v₁, ..., vₖ} 一定是子空間
- 是包含這些向量的**最小子空間**

### 範例

```
span{(1, 0)} = x 軸
span{(1, 0), (0, 1)} = ℝ²
span{(1, 2), (2, 4)} = 通過原點，斜率為 2 的直線
                     （因為 (2, 4) = 2(1, 2)）
```

## 8. 考試重點

### 常見題型

1. **判斷子空間**：給定集合定義，證明是/不是子空間
2. **找反例**：說明為何某集合不是子空間
3. **計算 span**：給定向量組，描述其張成

### 常見陷阱

- 忘記檢查零向量
- 只檢查有限個點，未考慮所有線性組合
- 混淆「子集」和「子空間」

### 快速判斷

```
不是子空間的信號：
- 「≥ 0」或「≤ 0」的條件
- 「= 常數（非零）」的條件
- 「整數」或離散條件
```
