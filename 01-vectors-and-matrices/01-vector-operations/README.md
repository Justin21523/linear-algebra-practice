# 向量運算 (Vector Operations)

> 線性代數的第一步：理解向量的本質與基本運算

## 1. 什麼是向量？(What is a Vector?)

### 三種觀點

| 觀點 | 說明 | 範例 |
|------|------|------|
| **物理觀點** | 有大小和方向的箭頭 | 力、速度、位移 |
| **電腦科學觀點** | 有序的數字列表 | `[3, 4]`、`[1, 2, 3]` |
| **數學觀點** | 向量空間中的元素 | 滿足向量加法和純量乘法的物件 |

### 向量表示法

```
列向量 (Column vector)：     行向量 (Row vector)：
    [3]
v = [4]                      v = [3  4  5]
    [5]
```

在線性代數中，**預設使用列向量**。行向量可用轉置表示：vᵀ = [3 4 5]

---

## 2. 向量加法 (Vector Addition)

### 定義

兩個**相同維度**的向量，對應分量相加：

```
u = [u₁]    v = [v₁]         [u₁ + v₁]
    [u₂]        [v₂]    u+v= [u₂ + v₂]
    [u₃]        [v₃]         [u₃ + v₃]
```

### 幾何意義：平行四邊形法則

```
        v
       ↗
      /
     /____→ u+v
    /    /
   /    /
  •----→ u
```

將向量 v 的起點移到向量 u 的終點，u+v 就是從原點到 v 終點的向量。

### 性質

1. **交換律** (Commutative): u + v = v + u
2. **結合律** (Associative): (u + v) + w = u + (v + w)
3. **零向量** (Zero vector): v + 0 = v
4. **反向量** (Inverse): v + (-v) = 0

---

## 3. 純量乘法 (Scalar Multiplication)

### 定義

純量（scalar，即一般的數字）乘以向量，每個分量都乘以該純量：

```
      [v₁]       [cv₁]
c ·   [v₂]   =   [cv₂]
      [v₃]       [cv₃]
```

### 幾何意義

- **c > 1**：向量伸長
- **0 < c < 1**：向量縮短
- **c < 0**：向量反向
- **c = 0**：變成零向量

```
原向量 v:     →→→→
2v:           →→→→→→→→
0.5v:         →→
-v:           ←←←←
```

### 性質

1. **分配律** (Distributive): c(u + v) = cu + cv
2. **分配律**: (c + d)v = cv + dv
3. **結合律**: c(dv) = (cd)v
4. **單位元**: 1·v = v

---

## 4. 線性組合 (Linear Combination)

### 定義

給定向量 v₁, v₂, ..., vₙ 和純量 c₁, c₂, ..., cₙ，

**線性組合** = c₁v₁ + c₂v₂ + ... + cₙvₙ

### 重要性

線性組合是線性代數的核心概念：
- **向量空間的生成**：所有線性組合形成的集合
- **線性相依/獨立**：是否能用其他向量的線性組合表示
- **基底**：最小的生成集合

### 範例

```
v₁ = [1]    v₂ = [0]
     [0]         [1]

任何 2D 向量都可以表示為 v₁ 和 v₂ 的線性組合：
[3]   = 3·[1] + 4·[0] = 3v₁ + 4v₂
[4]      [0]     [1]
```

---

## 5. 向量長度 (Vector Norm / Magnitude)

### 定義

向量的**長度**或**範數** (norm)：

```
‖v‖ = √(v₁² + v₂² + ... + vₙ²)
```

### 2D 和 3D 範例

```
v = [3, 4]      →  ‖v‖ = √(9 + 16) = √25 = 5
v = [1, 2, 2]   →  ‖v‖ = √(1 + 4 + 4) = √9 = 3
```

### 單位向量 (Unit Vector)

長度為 1 的向量稱為**單位向量**：

```
û = v / ‖v‖
```

這個過程稱為**正規化** (normalization)。

---

## 6. 內積 (Dot Product / Inner Product)

### 定義

```
u · v = u₁v₁ + u₂v₂ + ... + uₙvₙ
```

也可以寫成矩陣形式：u · v = uᵀv

### 幾何意義

```
u · v = ‖u‖ ‖v‖ cos(θ)
```

其中 θ 是兩向量的夾角。

### 重要應用

| 內積值 | 幾何意義 |
|--------|----------|
| u · v = 0 | u ⊥ v（正交/垂直）|
| u · v > 0 | 夾角 < 90° |
| u · v < 0 | 夾角 > 90° |

### 投影 (Projection)

向量 u 在向量 v 上的投影：

```
proj_v(u) = (u · v / ‖v‖²) · v
```

投影長度（純量）：

```
comp_v(u) = u · v / ‖v‖
```

## 8. 考試重點

### 常見題型

1. **計算題**：給定向量，求內積、長度、夾角
2. **證明題**：證明正交性（u · v = 0）
3. **應用題**：投影計算、單位向量

### 關鍵公式

```
‖v‖² = v · v                    # 長度平方等於自己與自己的內積
‖u + v‖² = ‖u‖² + 2(u·v) + ‖v‖²  # 平行四邊形法則
|u · v| ≤ ‖u‖ ‖v‖               # Cauchy-Schwarz 不等式
```

### 常見錯誤

- 混淆內積（結果是純量）和向量乘法
- 忘記檢查向量維度是否相同
- 投影公式分母忘記平方
